# Курсовая работа: Резервное копирование котиков 🐱

Программа для автоматизации скачивания изображений котиков с текстом и их последующей загрузки в облачное хранилище Яндекс.Диск.

## Функционал
- Скачивание случайных котиков с кастомным текстом через API [CATAAS](https://cataas.com).
- Автоматическое создание папки на Яндекс.Диске через API.
- Загрузка изображения в указанную папку.
- Сохранение отчета о загрузке в файл `upload_info.json`.

## Как использовать
1. **Получить токен**: Перейдите в [Яндекс.Полигон](https://yandex.ru) и получите OAuth-токен.
2. **Запустить скрипт**:
   ```bash
   python main.py
   ```
3. **Введите данные**:
   - Текст, который будет написан на котике.
   - Ваш OAuth-токен.
   - Имя папки, которая будет создана на диске.

## Технологии
- Python 3
- Requests (для работы с API)
- JSON (для формирования отчетов)
