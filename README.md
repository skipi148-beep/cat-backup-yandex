# cat-backup-yandex
# Программа резервного копирования изображений на Яндекс.Диск

Курсовая работа по курсу Python-разработчик. Программа позволяет скачивать изображения котиков с сервиса CATAAS и автоматически загружать их в облачное хранилище Яндекс.Диск.

## Возможности
- Получение изображения с наложенным пользовательским текстом.
- Автоматическое создание отчета в формате JSON с данными о загруженном файле.
- Работа с API Яндекс.Диска через протокол HTTP.

## Инструменты и технологии
- **Python 3.14**
- **Библиотека urllib**: используется для отправки запросов к API.
- **API Яндекс.Диска**: для загрузки файлов в облако.
- **API CATAAS**: для генерации изображений.

## Как запустить программу
1. Клонируйте репозиторий.
2. Получите OAuth-токен Яндекс.Диска на [Полигоне Яндекс.Диска](https://yandex.ru).
3. Запустите скрипт:
   ```bash
   python main.py
   ```
4. Следуйте инструкциям в консоли:
   - Введите текст, который будет на картинке.
   - Введите ваш OAuth-токен.
   - Введите название папки на Диске (папка должна быть создана заранее).

## Результат работы
- Выбранное изображение сохраняется на Яндекс.Диске в указанной папке.
- В корне проекта создается файл `upload_info.json`, содержащий имя файла и его размер в байтах.
